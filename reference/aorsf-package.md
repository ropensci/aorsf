# aorsf: Accelerated Oblique Random Forests

Fit, interpret, and compute predictions with oblique random forests.
Includes support for partial dependence, variable importance, passing
customized functions for variable importance and identification of
linear combinations of features. Methods for the oblique random survival
forest are described in Jaeger et al., (2023)
[doi:10.1080/10618600.2023.2231048](https://doi.org/10.1080/10618600.2023.2231048)
.
